Aplicação: Sistema de Gerenciamento de Produtos


Tecnologias Utilizadas:

Python
SQLAlchemy (para interação com o banco de dados)
SQLite (banco de dados)


Objetivo da Aplicação:
O objetivo desta aplicação é fornecer um sistema de gerenciamento de produtos com funcionalidades de CRUD (Create, Read, Update e Delete) para manipulação dos dados. A aplicação permite cadastrar, visualizar, atualizar e excluir produtos do estoque.
